关于本站

这里是卡戎Charon的个人网站。

这里会不定期更新博客文章、碎碎念（短动态）和一些小工具，希望这些东西有所帮助。如页脚所言，这些内容全部进入公有领域，任何人都可以随意使用这些内容，包括但不限于分享、商用、修改、训练AI。无需署名，无需询问。

不过，如果愿意，还请在引用到时告诉我一声。这不是强制的，只是为了满足卡戎对于「究竟有多少人用到了这个网站」这种事的好奇心。

网站里可能有些页面包含裸露、政治敏感、AI生成的内容，但会有所提示。

以下是技术细节。

这个网站只用到了前端三件套（HTML + CSS + JS），没有任何框架。同时，这个网站是完全静态的，没有后端。理论上，甚至可以在断网状态下靠缓存继续浏览这个网站。

你可能会好奇，既然没有后端，评论功能是如何运作的？事实上，评论内容储存在Nostr网络上。Nostr是一个多中心网络，评论会被同时储存在多个中继服务器（Relay）中。
